On-demand Design Service

【颀观芯片】致力于为客户提供丰富多样、灵活高效的芯片设计服务ODS，即On-demand Design Service，从前端到后端、从逻辑设计到物理设计、从功能设计到可测性设计、从新IP开发到全芯片SoC设计、从ASIC到FPGA，几乎可以涵盖芯片开发的全部流程。

【颀观芯片】可以为您提供以下芯片开发流程中任何阶段的服务内容，例如：

• 如您有新IP开发的需求，可委托【颀观芯片】进行Phase A阶段的开发。

• 如您有后端设计需求，可委托【颀观芯片】进行Phase B阶段的开发。

• 您可以单独选择芯片可测性设计服务：Phase B-2(DFT)。

您还可以选择针对timing相关服务内容：Phase B-1(Synthesis)以及Phase B-3.2(STA)。
您还可以委托【颀观芯片】进行ASIC全芯片定制设计，即选择Phase A+B。

Phase A :: LD，aka Logical Design

Phase B :: DI, aka Design Implementation

Phase C :: SV, aka Silicon Validation

以上仅仅是部分典型的ODS服务内容。无论您在芯片产品开发过程中需要什么样的帮助，欢迎您通过下面的方式联系我们，万一我们真的能帮上忙呢！
